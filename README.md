# Social Media Sentiment Analysis Dashboard

## Project Overview
This project analyzes customer sentiment from social media comments, reviews, and posts using Natural Language Processing (NLP) and Machine Learning.

It classifies text into:
- Positive 😊
- Negative 😡
- Neutral 😐

A Streamlit dashboard is used to visualize sentiment insights.

---

## Features
- Text cleaning and preprocessing
- Stopword removal
- TF-IDF feature extraction
- Logistic Regression model
- Live sentiment prediction
- Positive / Negative / Neutral classification
- KPI cards
- Bar chart visualization
- Pie chart visualization
- Word Cloud
- CSV upload for bulk sentiment analysis
- Download analyzed results

---

## Tech Stack
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib
- Streamlit
- Joblib
- WordCloud

---

## Model Performance
- Model: Logistic Regression
- Feature Extraction: TF-IDF
- Accuracy: 77.5%
